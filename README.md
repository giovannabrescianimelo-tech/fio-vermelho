# FioVermelho

Projeto desenvolvido com foco na aplicação prática de conceitos básicos de HTML, explorando estruturação de páginas e organização de conteúdo. Iniciativa voltada ao aprendizado e experimentação em desenvolvimento web.

## Objetivo
Aplicar na prática os conhecimentos introdutórios de HTML adquiridos em aula, criando uma página simples.

## Funcionalidades
- Estruturação básica de página web  
- Organização de conteúdo em HTML  
- Criação de layout simples  

## Tecnologias utilizadas
- HTML

## Aprendizados
- Estrutura básica de um documento HTML  
- Uso de tags e organização de conteúdo  
- Desenvolvimento de páginas estáticas  

---

Projeto com fins educacionais.
